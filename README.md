# Tech-Stand

My project is a multi-use tech stand! It can hold my phone both vertically and horizontaly, making it useful in many different situations. At the back of the tech stand, there is a small opening designed to hold my airpods at the same time. There are a few reasons why I wanted to make this project, with the first being that I don't have a phone stand, and the second being that I wanted to learn more about how to use fusion. I learned many new things when designing the stand, such as what constraints are, how to use them, and why they are so useful.
